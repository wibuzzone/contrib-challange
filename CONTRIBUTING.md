# 📖 Panduan Berkontribusi & Mengikuti Event

Terima kasih ingin ikut serta di **Wibuzone Open Contribution Challenge**! Ikuti langkah berikut agar karya kamu bisa kami terima dan dinilai:

---

## 📁 Aturan Penyimpanan Proyek
Silakan masukkan proyek ke folder yang sesuai:
- Proyek **JavaScript** → `projects/javascript/nama-proyek-kamu/`
- Proyek **Node.js** → `projects/nodejs/nama-proyek-kamu/`
- Proyek **PHP** → `projects/php/nama-proyek-kamu/`

Setiap folder proyek WAJIB berisi:
- Semua file kode yang dibutuhkan
- File `README.md` di dalam foldernya sendiri yang berisi:
  • Nama proyek
  • Deskripsi singkat fungsi proyek
  • Cara instalasi / cara menjalankannya
  • Nama & akun GitHub pembuat

---
